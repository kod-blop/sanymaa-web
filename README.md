# Sanymaa Social Hub

Bu proje, Sanymaa'nın tüm sosyal medya hesaplarını tek bir noktada toplayan, **"Hard Shake"** ve **"Deep Zoom"** animasyonlarıyla güçlendirilmiş, modern ve karanlık temalı bir profil sayfasıdır.

## 🚀 Özellikler
- **Modern Tasarım:** Karanlık mod ve neon renk geçişleri.
- **Akıcı Animasyonlar:** 
  - Profil fotoğrafı için pürüzsüz `Deep Zoom` girişi.
  - Butonlar üzerinde etkileşimli `Hard Shake` sarsıntı efekti.
- **Responsive:** Mobil ve masaüstü cihazlarla tam uyumlu.
- **Tek Sayfa:** HTML, CSS ve ikon kütüphaneleri tek bir `index.html` dosyasında optimize edilmiştir.

## 🛠 Kullanılan Teknolojiler
- HTML5
- CSS3 (Keyframe Animasyonları)
- FontAwesome (İkonlar)
- Google Fonts (Poppins)
